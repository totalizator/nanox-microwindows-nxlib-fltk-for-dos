### FlWriter is a text processing program using FLTK as a GUI tookit. It was developed by Jean-Mark Lienher in 2002 and ported to DOS and enhanced in 2012 by Georg Potthast. ###

### FlWriter uses XHTML as its internal format and stores the text files in this format. It features WYSIWYG . The editor window displays the text in the selected fonts. The document then can be converted for printing into a postscript file. Using Ghostscript you can also generate PDF documents and print files for HP Deskjet and Laserjet printers as well as Canon Bubblejet printers. ###
